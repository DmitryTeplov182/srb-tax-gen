# Генератор платежек с QR-кодами для Сербии

Веб-сервис для генерации платежек с QR-кодами. Автоматически заполняет назначение платежа предыдущим месяцем/годом. Использует API NBS IPS QR. Вся обработка происходит в браузере - данные не сохраняются на сервере.

> **Примечание:** Код этого проекта был написан с помощью нейросети (AI).

## Описание

Сервис позволяет удобно генерировать платежки на оплату налогов и взносов в Сербии:
- Допринос за ПИО
- Допринос за ЗДРО
- Допринос за НЗС
- Порез на паушални приход

## Особенности

- Автоматическая генерация QR-кодов через API Национального банка Сербии
- Автоматическое заполнение назначения платежа предыдущим месяцем и годом
- Все данные хранятся только в браузере (localStorage)
- Возможность сохранения и загрузки данных в файл
- Поддержка всех типов платежек с дефолтными суммами
- Не требует серверной части - работает полностью в браузере

## Использование

1. Откройте `index.html` в браузере
2. Выберите тип платежки
3. Заполните необходимые поля (некоторые заполняются автоматически)
4. QR-код генерируется автоматически
5. Используйте кнопки для сохранения/загрузки данных

## Технологии

- HTML5
- CSS3
- Vanilla JavaScript
- NBS IPS QR API

## API

Сервис использует официальный API Национального банка Сербии:
- Endpoint: `https://nbs.rs/QRcode/api/qr/v1/generate`
- Метод: POST
- Формат данных: pipe-separated text

## Безопасность

- Все данные обрабатываются локально в браузере
- Никакие данные не отправляются на сторонние серверы (кроме API NBS для генерации QR-кода)
- Данные сохраняются только в localStorage браузера
- Рекомендуется периодически экспортировать данные в файл для резервного копирования
